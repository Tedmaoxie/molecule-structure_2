# AI赋能高中化学教学系列by实验中学 VSEPR VISUALIZER

**AI赋能高中化学教学系列 by 实验中学**

这是一个基于 Web 的 3D 可视化教学工具，旨在帮助高中学生直观理解 **价层电子对互斥理论 (VSEPR Theory)**。通过交互式的 3D 模型，用户可以探索不同分子的几何构型，观察孤对电子对键角的影响，并查看真实的实验数据。

## 🌟 主要功能

*   **交互式 3D 渲染**：利用 Three.js 技术，提供流畅的分子旋转、缩放和平移体验。
*   **丰富的分子库**：
    *   **经典分子**：CO₂, H₂O, NH₃, CH₄, PCl₅, SF₆ 等。
    *   **扩展分子**：SO₂, O₃, HCHO (甲醛), BF₃ 等。
    *   **离子**：NH₄⁺, CO₃²⁻, SO₄²⁻, NO₂⁻ 等。
*   **真实实验数据**：
    *   集成 NIST 及权威期刊的真实键长、键角数据。
    *   **动态修正**：针对 SO₂, O₃, NO₂⁻ 等分子，根据孤对电子斥力实时动态计算并展示真实的弯曲构型（如 SO₂ 键角约为 119.0°）。
    *   **HCHO 特殊处理**：精确展示甲醛分子的双键角结构 (H-C-H 116.5°, H-C-O 121.8°)。
*   **动态交互控制**：
    *   自由调节成键电子对数 (BP) 和孤电子对数 (LP)。
    *   **斥力强度调节**：通过滑块动态调整孤对电子的斥力强度，实时观察其对键角的“挤压”效果。
*   **构型分析 HUD**：实时显示分子的通式 (AXmEn)、电子域几何构型、分子空间构型及理想/真实键角。

## 📚 技术栈

*   **Three.js**: 核心 3D 渲染引擎。
*   **CSS2DRenderer**: 用于在 3D 场景中显示 HTML 标签（如键角数值）。
*   **TWEEN.js**: 实现分子结构切换时的平滑过渡动画。
*   **HTML5/CSS3**: 构建现代化的用户界面 (UI)。
